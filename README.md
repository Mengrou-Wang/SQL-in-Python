# SQL-in-Python

The purpose of this project is to provide recommendations for a video shop to increase revenues. Using the public SQL database, key recommendations are:

- Overall, keep maintaining a sufficient inventory for sports as this is a very profit category in both stores.
- Try to find more drama, comedy and news movies for Store 1 to replace some action, family and animation movies.
- Collect more games and sci-fi movies for Store 2 to replace some family copies.
